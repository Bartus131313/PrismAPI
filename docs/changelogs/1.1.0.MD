# 📄 Change Log - `v1.1.0` (TBD)
> Expanding the API into the third dimension. Introducing a robust 3D-on-2D rendering pipeline with full camera control.
> 
## 🎥 3D Rendering Pipeline
* Coordinate Mapping: Introduced a 3D-to-2D grid system for placing blocks and structures within a standard GUI context.
* Universal Block Rendering: Direct support for BlockState and BlockEntity (Custom Model) rendering with automatic centering.
* Multi-Part Piston Logic: Specialized helper for animating Piston extension/retraction with proper model state swapping (hiding/showing the head).
* Decal Overlay: Support for rendering Breaking Textures (cracks) over 3D models using the SheetedDecalTextureGenerator.

## 🏗️ Structure & Geometry
* NBT Structure Support: (New) Ability to load and render .nbt structure templates as a 3D preview within the GUI.
* Boundary Outlines: (New) High-visibility 3D bounding box rendering for block selection and highlights.

## 🔭 Camera & Viewport
* Projection Matrix: Integrated enableCamera and disableCamera logic to handle 3D perspective, rotation (X/Y), and FOV scaling.
* The "Prism Hook": Introduced renderCustom3D, a functional interface allowing developers to inject their own rendering logic into the Prism 3D space.
