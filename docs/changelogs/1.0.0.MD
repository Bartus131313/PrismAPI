# 📄 Change Log - `v1.0.0` (28.03.2026)
> Initial Release: The core foundation for advanced 2D GUI manipulation and dynamic texture rendering.

## 🖼️ Advanced Rendering
* Fluid Engine: Native support for rendering animated fluid textures and dynamic fluid tanks (includes integrated tooltips).
* Progress Systems: Highly customizable vertical/horizontal fill bars supporting both solid colors and textured sprites.
* Text Suite: Enhanced string rendering with support for linear gradients, automatic centering, and shadow control.
* Scissor Stack: Simplified Start/End scissor implementation for clean UI masking and overflow clipping.

## ⚡ Animation & Layout
* Prism Animator: A lightweight interpolation engine for smooth UI transitions and element scaling.
* Prism Layout: A structured alignment system for managing complex GUI element positioning.

## 🛠️ Utilities
* Color Math: Utility for dynamic Alpha-blending and hex-color manipulation.
* Unit Formatting: Standardized number formatting for large values (K, M, B).
